1. Discuss the scope of var, let, and const

var:

  1. variable declared with var can have global and local scope.
  2. var can be declared outside the function.
  3. var can also be declared inside the function.

let:

  1. variable declared with let can have global ,local and block scope.
  2. let variable can be declared both inside and outside the function.
  3. let can be used in block scope where the value of let can be redeclared and reassigned.

const:

  1. const is used to declaring constant variable.
  2. const variable can be declared both inside and outside the function
  3. It cannot be reassigned and redeclared with other values.

2. Tell us the use cases of null and undefined

Undefined

  1. Variable that is not initialized will give undefined
  2. function with no return
  3. parameter that is not passed will return undefined
  4. if return has nothing on the right side will return undefined
  5. deleting an element inside an array

null

  1. when the variables or properties with no initial value.
  2. To represent that a function or method does not return any meaningful result.

What do you mean by REST API?

An API is a collection of resources and HTTP methods are rooted under a common url. Rest API receive HTTP request from clients and does whatever request needs. Some rest API https methods:

  1. GET: It provides read data
  2. POST: It is used to create a new database.
  3. PUT: It is used to update a new database.
  4. DELETE: It is used to remove a database.
